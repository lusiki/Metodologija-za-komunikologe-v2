# Metode istraživanja masovne komunikacije

**Metodološke osnove istraživanja masovne komunikacije u komunikologiji, sociologiji i drugim društvenim znanostima**

Ivan Burić & Luka Šikić

---

## O knjizi

Ova knjiga sveučilišni je udžbenik koji na sustavan način uvodi studente u metode empirijskih društvenih istraživanja s naglaskom na područje masovne komunikacije. Knjiga pokriva čitav istraživački ciklus, od epistemoloških temelja i nacrta istraživanja, preko metoda prikupljanja podataka (anketa, analiza sadržaja, kvalitativne metode, mješovita metodologija), do statističke analize podataka i suvremenih računalnih metoda (analiza teksta, analiza mreža). Završni dio knjige obrađuje etiku istraživanja, valjanost i pouzdanost te budućnost komunikologije kao računalne društvene znanosti.

Iako je prvenstveno namijenjena studentima komunikacijskih znanosti, knjiga će biti korisna i studentima sociologije, politologije, pedagogije, informacijskih znanosti te svima koji žele razumjeti, primijeniti ili unaprijediti svoje metodološko znanje. Svi primjeri i podatci kontekstualizirani su u hrvatskom medijskom prostoru.

## Sadržaj

| Dio | Poglavlje | Tema |
|-----|-----------|------|
| | Uvod | Uvodne napomene |
| **I. Temelji znanstvenog istraživanja** | 1. Znanost i znanstvena spoznaja | Ciljevi znanosti, pseudoznanost, razvoj komunikologije |
| | 2. Epistemološke osnove istraživanja | Kvantitativne i kvalitativne spoznajne osnove |
| | 3. Nacrt istraživanja | Hipoteze, varijable, mjerne ljestvice, uzorkovanje |
| **II. Metode prikupljanja podataka** | 4. Anketa | Povijest, vrste, izrada upitnika, pogreške |
| | 5. Analiza sadržaja | Tradicionalni mediji, digitalno okružje, pouzdanost |
| | 6. Kvalitativne metode | Etnografija, diskurs, intervju, fokus grupe |
| | 7. Mješovita metodologija | Integracija kvantitativnih i kvalitativnih pristupa |
| **III. Analiza podataka** | 8. Deskriptivna statistika | Mjere centralne tendencije, varijabilnosti, vizualizacija |
| | 9. Osnove statističkog zaključivanja | Intervali pouzdanosti, testiranje hipoteza |
| | 10. Statistički testovi u praksi | T-test, ANOVA, korelacija, regresija, hi-kvadrat |
| **IV. Računalne metode** | 11. Računalna analiza teksta | Priprema podataka, reprezentacija teksta, pristupi analizi |
| | 12. Analiza mreža | Centralnost, struktura zajednica, primjene |
| **V. Okvir i perspektive** | 13. Etika znanstvenih istraživanja | Etička načela, privatnost, algoritamska pristranost |
| | 14. Valjanost i pouzdanost | Koncepti, kriteriji kvalitete, replikacijska kriza |
| | 15. Računalna društvena znanost | Epistemološke implikacije, nove metode, budućnost |

## Tehnički detalji

Knjiga je napisana u [Quarto](https://quarto.org/) sustavu za tehničko i znanstveno izdavaštvo. Izvorni tekst nalazi se u `.qmd` datotekama, a knjiga se može kompilirati u HTML, PDF i DOCX formate.

### Preduvjeti

- [Quarto](https://quarto.org/docs/get-started/) (>= 1.3)
- [R](https://cran.r-project.org/) (>= 4.2) za poglavlja s primjerima koda

### Kompiliranje knjige

```bash
quarto render
```

Kompilirana knjiga nalazit će se u direktoriju `_book/`.

Za pregled tijekom pisanja:

```bash
quarto preview
```
